# How to use

*Required** docker to run app
 
## Command

use comand in same folder of `docker-compose.yml`

start first time
```bash
docker compose up -d
```

stop
```bash
docker compose stop
```

resume
```bash
docker compose start
```

destroy
```bash
docker compose down
```

## use app

import api collection to postman

use api in seq. of `scrap`, `mcost-cnc`, `price-matrix`
